## Overview

Demonstrates Carbon DPI protocol for **MSME CBAM compliance** — helping India's 30,000+ MSME exporters to the EU generate verified emissions data required under the EU Carbon Border Adjustment Mechanism.

### Use Case

EU CBAM requires importers to declare embedded emissions in imported goods. Indian MSME exporters need verified emissions data to continue exporting to the EU. Carbon DPI provides:

1. Standardised data collection via CDIF
2. Deterministic emissions calculation
3. GIC issuance as proof of verified emissions
4. Export as CBAM EU XML format
